HASHING

Introduction

A cryptographic hashing function takes data of arbituray length
as input and produces an output of a fixed length known as a message digest or hash value.
Bitcoin utilzes a specific flavor of cryptographic hashing function known as the SHA2-256.

SHA2-256

The SHA2-256 is considered a one way hashing function. The ouput that is produced can be used to
reverse the algorithm and aquire the input. This concept of SHA2-256 being a one way function aids in the security of bitcoin transactions.
