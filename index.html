Summary

In this simple website, we will provide educational an analytical explanations of three cornerstone aspects of Bitcoin technology: The Merkle Tree, Hashing Functions, and the Consensus Algorithm.

Using examples and providing context we will illustrate how the technology works using easy to follow examples and interactive content.

We will also provide guidelines on how to get started working with Merkle Trees, Hashing, and Consensus using JavaScript, Python, and Java. See our repo page for the source code.
